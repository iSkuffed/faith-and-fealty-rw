# Faith & Fealty

Splits RimWorld's Ideology DLC into two independent systems on every pawn: **Religion** (spiritual/supernatural beliefs, deities) and **Ideology** (social/political/economic values).

## Features
- **Dual belief system** — every pawn has both a Religion and an Ideology with separate certainty, roles, abilities, and UI
- **4-step wizard** — replaces the vanilla ideoligion screen with Religion Memes → Religion Customize → Ideology Memes → Ideology Customize
- **Certainty & cognitive dissonance** — randomized erosion, conversion/reassurance mechanics, Agnostic fallback at 0%
- **Dual leadership** — religious and ideological leaders with unified cooldowns
- **Deity system** — named deities with grammar-based generation for structure memes
- **Data-driven presets** — XML-defined templates for NPC factions and players, no C# changes needed
- **Full save/load** — custom IExposable structs for complete persistence

## Requirements
- RimWorld 1.6, Harmony, Ideology DLC

## Compatibility
- Vanilla Ideology Expanded - Memes and Structures
- Alpha Memes
- RimHUD
