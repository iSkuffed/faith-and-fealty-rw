## Bug Reports
Submit via GitHub Issues. Must include:
- Full Player.log (use "Copy" in RimWorld's debug logger)
- Clear reproduction steps
- Complete mod list

Issues missing any of these will be closed immediately.

## Compatibility Patches
The mod uses XML patching — make a patch against the existing architecture and submit a PR.
If you're unable to make a patch, fork the repo for personal use. Please don't reupload the mod in full.
Contributors are credited. If your patch breaks and needs significant repair I may remove it (and your credit) to keep the project clean.

## Code Submissions
- Open an issue first to discuss changes
- **Source only** — no binaries/.dlls. PRs with binaries rejected on sight.
- I handle all compilation. Only source code will be accepted.
- PRs must include a clear description, rationale, and test instructions.
- Code should be clean and consistent with the existing style.
